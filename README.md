# n-bits

Algorithms to compress ML weights.


## Installation

```bash
pip install n_bits
```

## Usage

```bash
n-bits compress
n-bits error
n-bits expand
```

### As a Python Package

```python
from n_bits import my_function

print(my_function())
```
